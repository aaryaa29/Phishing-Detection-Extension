# Phishing-Detection-Extension
This project is an AI-powered phishing email detection system that classifies emails as **Safe** or **Phishing** using Machine Learning.   It is integrated with a **Chrome Extension** to allow users to analyze emails in real-time.

Dataset link- https://www.kaggle.com/datasets/subhajournal/phishingemails?resource=download

Create a new folder in your project called extensions and add this 4 files under it.
1. manifest.json
2. popop.html
3. popup.js
4. style.css

Phishing detection project/

Phishing_Email.csv # Dataset
train_model.py # Model training script
model.pkl # Trained ML model
vectorizer.pkl # TF-IDF vectorizer
app.py # Flask API

 extension/

manifest.json
popup.html
popup.js
style.css

 Accuracy is of around **92%**
